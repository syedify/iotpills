NFC_HUB_edison.ino is provided with medicine, manufacturer, patient, doctor and pharmacist information and it is reponsible to updating the NFC tag with that information and initiating the upload to the database.
WiFi.ino initiates and ensure the wifi connection.
WiFiWebClient.ino connects and uploads the data to web server.
Main.ino polls for NFC tags and data to be written on the NFC tag.
